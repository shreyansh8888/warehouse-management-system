📌 Project Overview

The Warehouse Management System (WMS) is a desktop-based application developed using Python. It helps manage inventory, stock updates, sales records, and warehouse data efficiently.

The system allows users to:

Add and update products
Manage stock levels
Record sales transactions
View inventory details
Store and retrieve data using a database
🛠 Technologies Used
Python
CustomTkinter
SQLite
Matplotlib
🗄 Database

The application uses an SQLite database to store product and sales information.
It performs CRUD operations (Create, Read, Update, Delete) to manage warehouse data.

🚀 Features
Add new products
Update stock
Delete products
Record sales
Automatic stock deduction
Input validation
User-friendly interface
